ES5


						function(pizza, sliceNum) {
							var slices = [];
							for (var i = 0; i < sliceNum; i++) {
								slices.push(pizza.slice());
							}
							return slices;
						}

l'utilizzo ideale è quando si deve passare una funzione come argomento di un'altra funzione:


						orderPizza('Cheese', (pizza, totalCost) => {
							console.log('Ordered ' + pizza + ' for ' + totalCost);
						});
  • se l'argomento è solo uno allora le parentesi non sono necessarie
  • se l'espressione è solo una allora puoi eliminare le parentesi e il return statement
